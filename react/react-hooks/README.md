# React Hooks

## #️⃣React Hooks&#x20;

함수형은 클래스형에 비해 기능은 작지만 간결하고 빠르다.&#x20;

react 18.0이 출시되며 hooks가 기능이 작다는 단점을 보완해주었다.&#x20;

_\*Hooks: 함수형 컴포넌트에서 React state와 생명주기 기능을 연동 할 수 있게 해주는 함수_

사용 규칙

1. 최상위에서만 Hook을 호출 반복문, 조건문, 중첩된 함수 내에서 Hook을 실행하면 안된다. \
   → 렌더링될 때마다 항상 동일한 순서로 Hook이 호출되는 것이 보장
2. 리액트 함수 컴포넌트에서만 호출

### React Hooks 등장 계기

함수형 컴포넌트의 상태 관리 및 라이프사이클 사용 제약을 극복하기 위해 등장!

이전에는 클래스형 컴포넌트에서만 상태와 라이프사이클을 다룰 수 있었다. 함수형 컴포넌트는 단순히 함수로 정의되어 있어 상태와 라이프사이클 관리가 제한적이었다. 이러한 한계를 극복하기 위해 **React Hooks**가 등장했고, 함수형 컴포넌트에서도 상태와 라이프사이클을 다룰 수 있게 되었다.
